GetFloorPlane()

Is called to retrieve the plane equation of the floor.

public Plane GetFloorPlane()

Returns

Plane

The plane equation of the floor plane.

Remarks

The plane equation is used by the Navigation Library to determine the floor for the walk navigation mode, where the height of the eye is fixed to 1.5m above the floor plane. The floor need not be parallel to the world ground plane.

The Navigation Library assumes that this value does not change during navigation. Should it change then the Navigation Library needs to be notified what the new value is.

Exceptions

NoDataException

There is no selection.

NotImplementedException

The requested method or operation is not implemented.
